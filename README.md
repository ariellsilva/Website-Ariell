
# 💼 Portfólio Pessoal

Projeto desenvolvido com foco em apresentar meus trabalhos, habilidades e evolução na área de desenvolvimento web.

Este portfólio foi criado utilizando **HTML, CSS e JavaScript**, com estrutura organizada e código limpo, priorizando responsividade e interatividade.

---

## 🚀 Funcionalidades

### 💼 Sistema Dinâmico de Projetos

Aplicação desenvolvida em **JavaScript** que permite alternar entre diferentes projetos através de um botão interativo.

Ao clicar, o sistema atualiza dinamicamente:

* 📌 Título
* 🖼️ Imagem
* 📝 Descrição

Tudo em tempo real, proporcionando uma navegação fluida e moderna.

---

### 📱 Design Responsivo

Layout totalmente adaptado para diferentes dispositivos:

* 📱 Smartphones
* 📲 Tablets
* 💻 Desktops

Utilizando **@media queries**, o site mantém usabilidade e organização em qualquer tamanho de tela.

---

### 📩 Contato Direto

Formulário funcional integrado ao **FormSubmit**, permitindo envio de mensagens diretamente para meu e-mail sem necessidade de back-end.

---

## 📂 Estrutura do Projeto

```
📁 portfolio

 ├── style.css       # Estilização e responsividade
 ├── javascript.js   # Lógica de alternância dos projetos
 └── img/            # Imagens (certificados, projetos, fotos)
```
